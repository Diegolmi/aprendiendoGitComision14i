git clone

clonar repositorio Subir mis cambios

git add . ...... comienzo a preprar mis archivos para subirlos

git status ----- que cambios estoy preparando para subir

git commit -m "mensajes"

git push origin main ------ subir mis cambios al repositorio remoto

Actualizar repositorio local

git pull ---- Actualizar mi repositorio local

git pull origin main

Creacion de Branch

git checkout -b "nombre" ---- creamos una branch

git checkout "nombre de la branch" ---- navego entre las branchs

git branch -a ver todas branchs que estan en el repositorio remoto

git branch -d "nombre de la branch" ---- eliminar branch
